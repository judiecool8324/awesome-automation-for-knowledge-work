## 🔍 Understanding automation patterns

Automation patterns define how the software handles your files. You can choose from several pre-built templates:

*   Document Categorization: Automatically sort incoming email attachments into specific folders based on keywords.
*   Information Extraction: Pull specific data points, such as dates or invoice numbers, from PDF files into a spreadsheet.
*   Semantic Search Indexing: Create a searchable library of your documents. You can ask the software questions about your files to find answers quickly.
*   Flow Triggers: Send a notification to your email or messaging app when a specific file event happens.

## 💡 How to automate a workflow

To create your first automated task, navigate to the "Workflows" tab in the main menu:

1.  Click the "New Workflow" button.
2.  Select a "Trigger." This tells the computer what to watch, like a folder full of recent invoices.
3.  Select an "Action." This tells the computer what to do, like "Move to Archive" or "Send to Excel."
4.  Define your "Logic." If your workflow acts on specific document types, add a condition here.
5.  Save the workflow.
6.  Click the "Run" button to test the process.

The "Human-in-the-Loop" setting allows you to review the computer's work before it finalizes an action. Enable this setting if you prefer to approve changes manually.

## 🛡️ Privacy and security

The tool processes your files locally on your computer. Your sensitive data stays within your local file system. The software does not send your documents to external cloud services unless you specifically enable a cloud integration feature.

## 🔧 Troubleshooting common issues

If you encounter problems, check these items:

*   The software will not start: Restart your computer and try launching the application again. Ensure no other instances are running in the background via the Task Manager.
*   Permission denied errors: Ensure you have read and write access to the folders you want to automate.
*   File indexing seems stuck: Clear the application cache in the Settings menu and restart the scan.
*   Installer does not run: Check that you are logged in as a user with administrative rights.
